Build fast, responsive, and accessible photo galleries in WordPress.

== Description ==

MY Gallery PRO lets photographers, creative professionals, and content-rich WordPress sites build responsive image galleries from the native Media Library.

Create a gallery, choose and reorder images, select a responsive layout, then place it on any page or post with a shortcode. Images retain their WordPress responsive sources and alternative text, while the optional viewer works as a progressive enhancement over ordinary full-image links.

MY Gallery PRO is free and open source, released under the GNU General Public License v3.0. See the bundled LICENSE file for terms.

== Installation ==

1. Upload the official plugin ZIP from **Plugins > Add New > Upload Plugin**.
2. Activate MY Gallery PRO.
3. Open **MY Gallery PRO > Add Gallery**.
4. Choose Media Library images, configure the layout, and save.
5. Copy the generated `[mygallery id="123"]` shortcode into a page or post.

== Frequently Asked Questions ==

= Does uninstalling remove galleries or media? =

No. User content is preserved by default.

== Changelog ==

= 1.0.0 =

* Initial release of MY Gallery PRO — build fast, responsive, and accessible photo galleries in WordPress.
* Native WordPress Media Library gallery creation with image selection, keyboard-accessible ordering, captions, and live preview.
* Responsive Grid and Masonry layouts with separate desktop (1–6) and mobile (1–3) column controls, responsive to narrow theme and page-builder containers.
* Accessible first-party image viewer with keyboard navigation and a no-JavaScript full-image fallback.
* `[mygallery]` shortcode with one-click copying; images keep their WordPress responsive sources and alternative text.
* Multi-author gallery isolation, attachment access checks, bounded gallery lists, and directory-listing guard files.
